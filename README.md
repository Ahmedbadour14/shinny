# Shinny — Handmade Beaded Bags

Modern e-commerce platform for handmade beaded bags in Egypt.

## Tech Stack
- **Frontend**: React + Vite + TypeScript + Tailwind CSS + Framer Motion
- **Backend**: Node.js + Express + Prisma + SQLite
- **i18n**: Arabic / English with RTL support

## Quick Start
```bash
npm run install:all
cd server && npm run db:migrate && npm run seed
npm run dev
```
- Client: http://localhost:5173
- Server: http://localhost:3001
- Admin: http://localhost:5173/admin

## Environment Variables
Copy `server/.env.example` to `server/.env` and fill in your keys.
